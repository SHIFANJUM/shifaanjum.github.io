About the Project

Comprehensive Overview of SecureBank

SecureBank is a full-featured client-side digital banking simulation that demonstrates advanced web technologies combined with cryptographic techniques.

This project integrates multiple banking modules including ATM operations, UPI transactions, card management, admin analytics, and LSB steganography for secure transaction logging.

All operations are performed on the client-side using LocalStorage as the backend, ensuring a complete educational demonstration of modern banking systems with enhanced security through steganography.

System Architecture

Complete Component Overview

The system architecture integrates User Interface with Authentication Handler, separate modules for ATM, UPI, and Card operations, a powerful Steganography Engine for secure data hiding, and an Admin Dashboard for system monitoring. All data flows through LocalStorage Backend ensuring complete client-side operation.

Steganography Workflow

5-Step Process for Hiding Transaction Data in Images

1

JSON to Binary

Convert transaction JSON data into binary string representation

2

LSB Modification

Modify Least Significant Bits of RGB pixel values

3

Canvas Embedding

Embed modified pixel data into canvas image

4

Image Export

Download encoded image as PNG file

5

Data Recovery

Extract LSB from image and reconstruct original JSON
